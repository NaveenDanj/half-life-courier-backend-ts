# User Stories (next iteration)

1. As a client, I want to schedule recurring pickups so I can automate frequent shipments.
2. As an operator, I want route optimization suggestions for multiple pickups to reduce cost/time.
3. As a client, I want real-time GPS tracking of active deliveries to see exact ETAs.
4. As an admin, I want bulk import/export of shipments via CSV to speed onboarding.
5. As a client, I want printable shipping labels (PDF) generated from shipment data.
